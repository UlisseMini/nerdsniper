Nerdsniper is an effort started by me to make finding people easier, by leveraging publically accessable data from social media.

The idea is that in future you would be able to search things like this

chess OR coder age:15-18 type:intj gender:F

and nerdsniper would be able to search millions of profiles, automatically pulled from social media and present you with the best matches.

Currently most features are aspirational, but the prototype is already quite capable, despite it only using a text search through people's twitter bios! I have hope that with enough work I can get this to a point of working quite well

Here's a feature roadmap, in rough order of difficulty. excluding the huge number of boring changes never seen by users.

  1. Get prototype working
  2. Begin predicting gender
  3. Begin inferring interests from tweets and bio
  4. Begin predicting age
  5. Begin predicting myers briggs type
